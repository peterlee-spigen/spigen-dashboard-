# 조코딩 샘플 데이터 - 시간축 유지 저밀도 버전

- 목적: 원본의 최초 시점부터 최신 시점까지 시간축은 유지하면서, 각 시점의 행 밀도를 낮춘 수업용 경량 데이터입니다.
- 원칙: 컬럼명과 컬럼 순서는 원본 CSV와 동일하게 유지하고, 행만 줄였습니다.
- 방식: 파일별 실제 최초/최신 날짜 행을 반드시 포함하고, 중간 날짜/월 버킷을 균등 간격으로 남긴 뒤 주요 속성 희소값을 보강했습니다.
- 주의: 원본 자체가 2026년 3월 한 달치인 CSV는 더 긴 기간을 만들지 않았습니다. 존재하지 않는 과거/미래 행은 합성하지 않았고, 해당 원본의 2026-03-01~2026-03-31 축만 저밀도로 유지했습니다.
- 원본 XLSX는 이미 작아서 동일 파일을 복사했습니다.

## 사용 기준

- 수업 중에는 이 폴더를 우선 사용하면 로딩 시간을 줄이면서도 시간축 분석 흐름을 설명할 수 있습니다.
- 원본 전체 분포나 성능 비교가 필요할 때만 기존 `조코딩 샘플 데이터` 폴더를 사용합니다.
